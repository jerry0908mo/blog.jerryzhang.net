让Sublime 支持GB2312格式，
Sublime默认只支持UTF-8编码格式

需要安装插件 CovertToUTF8

File: /Users/jerrysher/Downloads/Fiy5SDEjl8xM9NHWXNnEAw3e-8wh.srt
Encoding: GB2312
Error: Codecs missing

Please install Codecs33 plugin (https://github.com/seanliang/Codecs33/tree/osx).
