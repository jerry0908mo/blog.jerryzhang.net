学习 Pro Git 一书的学习笔记。
采用教材：Pro Git 中文版

一、起步

关于版本控制

  • 本地版本的控制 rcs 保存并管理文件补丁（patch）
  • 集中化的版本控制系统 CVS SubVersion Perfoce， 有集中的服务器管理版本，本地每次获得最新版本代码。
  • 分布式版本控制系统

Git 历史

2005年一直免费试用的 分布式版本控制系统 BitKeeper 的商业公司停止了对 Linux 的免费授权, Linus 就带领社区撸了一个分布式版本管理系统。

git 基础概念

  • 文件快照，直接保存文件，而不是差异比较
  • 本地操作 可以不用连接网络仓库 疯狂提交代码
  • 时刻保证数据完整性

文件的三个状态

  • 已提交
  • 已修改
  • 已暂存

为什么娱乐明星的收入比伟大的科学家高那么多？

今天在微博里看到阑夕在自己的微博里面发布一条言论：

“为什么娱乐明星的收入比伟大的科学家高那么多？为什么有钱人一掷千金的办宴席却不把钱捐给饿肚子的穷人？为什么西欧偶出一次爆炸案件的关注人数要远高于中东每天都在发生死伤规模也更多的同类惨剧？”因为你们蠢啊。

其实这条微博只有6个字和一个句号是阑夕的观点。虽然看起来有点粗暴直接，但是不一定没有道理。

赚钱的多少，吸引眼球的多少其实很多时候是看给社会创造的价值多少，另外可能要给这个创造价值加上一个时间那就是当前。还有一个社会的概念，社会就是这个世界，这个世界上的人们。所以说到底还是大家的喜好决定而分配方式。

有上面微博中类似疑问的朋友们，或许是想说，为什么这个社会这么不公平？其实这问题如果你有真的思考过，你或许不能得到答案但是你肯定可以找到让自己不再问类似问题的借口或者说理由。

世界读书日说读书

发表于 2016-04-24   |   分类于 读书   |  


今天是第21个世界读书日，我也是通过各种软件上还有朋友圈知道的，想不知道有点难度。微信读书昨天就放出了活动，设计了10道题目测试用户阅读量，答完题目分享可以得到一本电子书。当然其他公司也有很多活动，比如简书在北京和上海有组织线下的换书活动。豆瓣也有活动推出征集大家最喜欢的读的图书。

4月23日，世界图书与版权日，俗称世界读书日，是1995年开始设立的。同时这一天也是和很多大文人有关的纪念日。当然有的忌日，有的是诞辰。每个节日都有自己的起源，我就看了一篇百度百科，大概知道了一些。就不复制粘贴了。 总之这个节日设立的目的大概有三点：一、让大家都享受读书的快乐，无论穷富，无论年龄，二、感谢和纪念『文人们』注意版权问题，三、让大家哟个日子可以在一起庆祝（一起促销）。
